# L01E01: Hello world
Vytvořte skript `hello_world.py`, který po spuštění vypíše `Hello world!`. Pozor, záleží na velikosti znaků a dalších detailech!

## Příklad výstupu
```
> python3 hello_world.py
Hello world!
```

## Lokální testování
Funkčnost řešení ověříte následujícím příkazem:

```bash
pytest tests.py
```
